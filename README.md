# DevOps QR Code Generator

A sample DevOps capstone application that generates QR codes from user-provided URLs. The project includes a **Next.js** front-end, a **FastAPI (Python)** backend that stores generated QR codes in **AWS S3**, and **Terraform** for provisioning and managing cloud infrastructure. It is designed to provide hands-on experience with core DevOps practices such as infrastructure as code, containerization, CI/CD, and monitoring.
